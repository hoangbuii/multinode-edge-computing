# multinode-edge-computing
## Install
### Master Node (linux)
```bash
wget https://raw.githubusercontent.com/hoangbuii/multinode-edge-computing/refs/heads/main/release/ecmcli-1.1-linux.zip
unzip ecmcli-1.1-linux.zip
chmod +x ecmcli
sudo mv ecmcli /usr/bin
ecmcli version
```
